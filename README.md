# FreeWay

A web-based route planner for traffic-free public transit lines (metro, tram, metrobus, Marmaray, etc.). Users can manage transit lines and stops, then find the least-transfer route between two stops.

## Tech Stack

- **Frontend**: Vanilla JavaScript (SPA)
- **Backend**: Node.js + Express
- **Database**: SQLite
- **Testing**: Jest
- **API Docs**: Swagger UI
